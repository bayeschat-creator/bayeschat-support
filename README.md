# BayesChat Support

Welcome to BayesChat support.

BayesChat is a local AI chat client for users who configure their own model providers. If you need help, have a question, want to report a bug, or would like to request a feature, please use one of the support options below.

## Contact

Email: bayeschat@gmail.com

GitHub Issues: https://github.com/bayeschat-creator/bayeschat-support/issues

We normally review support requests as soon as possible. Response time may vary depending on the issue and time zone.

## What to Include

When requesting support, please include:

- App version
- Device model
- iOS, iPadOS, or macOS version
- A clear description of the issue
- Steps to reproduce the issue, if available
- Screenshots or screen recordings, if helpful
- Any relevant error message, without sharing private API keys, access tokens, or secrets

## Common Help Topics

### Provider Setup

Make sure the provider name, base URL, API key, and model ID are entered correctly in the app settings. If a request fails, verify that the configured provider is reachable from your device.

### Local Data

BayesChat stores chats and settings locally on your device. If you uninstall the app, local app data may be removed by the operating system.

### API Keys

Your API keys are stored on your device. Do not include API keys, access tokens, or other secrets when contacting support.

### In-App Purchase

If the Pro unlock does not appear after purchase, try restoring purchases from the app's purchase screen. Purchases are processed by Apple's App Store.

## Privacy

BayesChat does not operate a separate account system or support server for chat content. For privacy questions, contact bayeschat@gmail.com.
